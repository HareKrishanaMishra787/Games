# 🎯 Wordle-Game (Tkinter)

> A simple Python Project using Tkinter.

---

## 🚀 About

- Guess the secret word within 6 tries.
- Color feedback after each guess (Green, Yellow, Grey).
- Adjustable word length (3–6 letters).
- High score tracking.
- Smooth GUI with mouse and keyboard support.

---

## ⚙️ Installation

```bash
git clone https://github.com/yourusername/wordle-tkinter-clone.git
cd wordle-tkinter-clone
pip install -r requirements.txt
python main.py
```

# 🐢 Turtle Crossing Game

A fun and simple arcade-style game where your goal is to help the turtle 🐢 cross a busy road filled with fast-moving cars 🚗 without getting squashed!

Built entirely in Python using the `turtle` graphics library, this game is a great project to understand object-oriented programming, animation, collision detection, and more!

---

## 🎮 Game Preview

> Dodge cars. Cross the road. Level up. Repeat!

---

## 🚀 Features

- 🕹️ Keyboard control: Use the `Up` arrow key to move the turtle forward
- 🚘 Cars with increasing speed as levels progress
- 🔄 Continuous game loop and real-time screen update
- 💥 Collision detection ends the game
- 🏆 Scoreboard displays level progression

---

## 🗂️ Project Structure

```bash
├── main.py             # Main game loop and event handling
├── player.py           # Player (turtle) movement logic
├── car_manager.py      # Car spawning, movement, and level control
├── scoreboard.py       # Score tracking and game over display
```
